# Raindrop Search Reports

This directory contains search reports generated by the CLI tool.

## Report Formats

### Text (.txt)
Plain text format for easy reading in any text editor.

```bash
python search_cli.py "query" --format txt
```

### Markdown (.md)
Markdown format with clickable links, perfect for viewing in GitHub or markdown readers.

```bash
python search_cli.py "query" --format md
```

### JSON (.json)
JSON format with complete metadata, ideal for programmatic processing.

```bash
python search_cli.py "query" --format json
```

## Examples

### Search for David Grusch articles
```bash
python search_cli.py "david grusch" --format md
```

### Search UAP articles with specific tags
```bash
python search_cli.py "UAP" --tags aliens extraterrestrial --format json
```

### Search within a specific collection
```bash
python search_cli.py "AI" --collection 12345 --format txt
```

### First page only (fast)
```bash
python search_cli.py "python" --first-page-only
```

## Notes

- Reports are automatically saved with timestamp
- Query name is sanitized for filenames
- All pages are fetched by default (can retrieve 1000s of bookmarks)
- Batch requests (10 concurrent by default) for fast retrieval
- Reports are gitignored to keep your data private
